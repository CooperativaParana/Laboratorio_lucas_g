Definir User Stories detalladas

Autenticación y Sesión

el analista debe poder iniciar sesión con username/password para acceder al sistema
el analista debe poder cerrar sesión de forma segura
Si el analista se equivoca mas de 3 veces al colocar datos se bloquea temporalmente el sistema

Menu y Navegación

el analista debe poder ver las tareas que puede ejecutar
el analista debe poder navegar fácilmente por el menu entre las diferentes secciones en base a sus tareas

--- Seccion de Analista Palinilogico ---

Gestión de Muestras Palinológicas

El analista deberia poder agregar una nueva muestra pasando a un formulario de datos y luego al contador
El analista deberia poder buscar muestras por número de registro o fecha (reporte)
El analista debe de ver el listado de todas las muestras asi como estadisticas y graficos de las mismas (Graficos y Consultas)

Análisis Palinológico - Contador - Formulario de Muestra

El analista debe buscar especies en el catálogo por nombre científico o común
El analista debe registrar especies encontradas en una muestra
El analista debe ingresar cantidad de granos por especie
El analista debe asignar marcas especiales (x, #, ##) a las especies
El analista debe que el sistema calcule automáticamente los porcentajes
El analista debe completar el análisis y marcar fecha de análisis

Análisis Fisicoquímicos

El analista quiero registrar un análisis fisicoquímico para un tambor
El analista quiero ingresar valores de color, humedad, pH, acidez, conductividad
El analista quiero buscar análisis fisicoquímicos por tambor o fecha

Consultas y Reportes

El analista quiero buscar análisis previos por diferentes criterios
El analista quiero ver el detalle completo de un análisis palinológico
El analista quiero ver el detalle completo de un análisis fisicoquímico
El analista quiero generar reporte PDF del análisis palinológico
El analista quiero generar reporte PDF del análisis fisicoquímico
